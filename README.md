An end-to-end **Data Analytics portfolio project** that ingests the raw
Titanic dataset, cleans and preprocesses it, generates rich visualizations,
serves an interactive Streamlit dashboard and produces a polished PDF +
Markdown report.

---

## 📌 Project Overview

The project demonstrates a full analytics workflow:

1. **Load** a real-world raw dataset (Titanic passenger data).
2. **Clean** it — missing values, duplicates, outliers (IQR), dtypes,
   inconsistent categoricals.
3. **Process** — feature engineering, filtering, grouping, aggregation.
4. **Visualize** — histograms, bar/pie/box/scatter/line charts, heatmap.
5. **Serve** an interactive **Streamlit dashboard** with KPIs, filters and
   a cleaned-dataset download.
6. **Report** — Markdown + PDF containing 10+ business insights.

## ⚙️ Installation

```bash
# 1. Clone the repository
git clone https://github.com/<your-username>/Data_Analytics_Project.git
cd Data_Analytics_Project

# 2. (Optional) create a virtual environment
python -m venv .venv
source .venv/bin/activate      # Windows: .venv\Scripts\activate

# 3. Install dependencies
pip install -r requirements.txt
```

## ▶️ How to Run

**Run the full pipeline** (cleaning → visualization → reports):

```bash
python main.py
```

**Launch the interactive dashboard**:

```bash
streamlit run dashboard/dashboard.py
```

## 💡 Business Insights (excerpt)

- Females survived at ~74% vs males at ~19% — sex is the strongest predictor.
- First-class survival rate (~63%) is roughly 2.5× third-class (~24%).
- Fare correlates positively with survival (~0.26).
- Passengers with a recorded cabin survived at ~67% vs ~30% without.
- Traveling with family improved survival vs traveling alone.

Full list in [`reports/project_summary.md`](reports/project_summary.md).
